# LegalTech – Automated Contract Parsing & Risk Extraction Engine

## Team Members

* Jatin Raghuwanshi

## Project Overview

This project automates legal contract review by extracting text from PDF documents, identifying important legal entities, classifying legal clauses, and detecting high-risk provisions using Natural Language Processing (NLP).

* Django
* Django REST Framework (DRF)
* PostgreSQL
* PyMuPDF
